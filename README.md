# Automação de Solicitação (chamado) de Suporte Técnico (Power Platform)

## 📖 Descrição

Desenvolvimento de uma solução automatizada para gerenciamento de chamados de suporte técnico de T.I utilizando Power Platform. O projeto simula um fluxo de atendimento, desde a abertura do chamado até a notificação dos responsáveis.

---

## 🎯 Objetivo

Criar um sistema simples e automatizado para registro e acompanhamento de solicitações de suporte técnico, otimizando a comunicação entre usuário, gestor e equipe responsável.

---

## ⚙️ Funcionalidades

* Registro de chamados por meio de aplicativo
* Armazenamento das solicitações em lista do SharePoint
* Envio automático de e-mails com informações do chamado
* Notificação para:

  * Usuário solicitante
  * Gestor da área
  * Técnico responsável
* Simulação de fluxo de atendimento automatizado

---

## 🛠️ Tecnologias Utilizadas

* Power Apps
* Power Automate
* SharePoint

---

## 🧠 Minha Atuação

* Criação e estruturação do aplicativo no Power Apps
* Integração com lista de dados no SharePoint
* Definição e implementação da lógica de automação no Power Automate
* Organização do fluxo de envio de notificações
* Estruturação do processo de atendimento automatizado

O projeto permitiu simular um processo real de atendimento técnico automatizado, melhorando a organização das solicitações e garantindo comunicação eficiente entre as partes envolvidas.
